BaKa Plan

This software creates seating plan for examinations. It provides four different strategies to create seat plan.

REQUIREMENTS:

1) GNU G++ Compiler

2) Configure public_html/cgi-bin folder for executing files on browser.

3) CGICC Library
Download 3rd last package from ftp://ftp.gnu.org/gnu/cgicc/

Then run following commands in terminal

1) $tar xzf cgicc-X.X.X.tar.gz

2) $cd cgicc-X.X.X/

3) $./configure –prefix=/usr

NOTE: If you got permission error then use sudo with command.

4) $make

5) $make install
