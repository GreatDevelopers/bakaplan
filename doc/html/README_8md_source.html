1 BaKa Plan
2 =========
3 
4 This software creates seating plan for examinations. It provides five
5 different strategies to create seat plan.
6 
7 REQUIREMENTS:
8 -------------
9 
10  1) GNU G++ Compiler
11  2) Configure public_html and cgi-bin in home
12  3) CGICC Library
13  4) MySQL Connector for C++
14  5) jwSMTP Library
15  6) exim4 mail server
16  7) Enable site(localhost/bakaplan) for BaKaPlan

18 Installation of requirements
19 
20 1) GNU G++ Compiler
21 
22 Run following command in terminal to install
23 
24  $ sudo apt-get install g++
25 
26 2) Configure public\_html/cgi-bin folder for executing files on browser.<br>
27 Assuming you already installed apache if not then run following
28 command in terminal
29 
30  $ sudo apt-get install apache2
31 
32 **Steps to configure public_html**
33 
34  $ mkdir ~/public_html
35 
36  $ sudo a2enmod userdir
37 
38  $ sudo service apache2 restart
39 
40 Give 755 permissions to public\_html directory
41 
42  $ chmod -R 755 ~/public_html
43 
44 Now open http://localhost/~username in browser.
45 Here username is your login name.
46 
47 **Steps to configure cgi-bin in public_html**
48 
49  $ sudo a2enmod cgi
50 
51  $ sudo a2enmod cgid
52 
53  $ sudo service apache2 restart
54 
55  $ cd ~/public_html
56 
57  $ mkdir cgi-bin
58 
59  $ cd /etc/apache2
60 
61  $ sudo vim sites-available/default
62 
63 Add following text in file:
64 
65  ScriptAlias /cgi-bin/ /home/*/public_html/cgi-bin/
66  <Directory "/home/*/public_html/cgi-bin">
67  AllowOverride None
68  Options +ExecCGI -MultiViews +SymLinksIfOwnerMatch
69  SetHandler cgi-script
70  Order allow,deny
71  Allow from all
72  </Directory>
73 
74 Save it and then restart apache
75 
76  $ sudo service apache2 restart

129 6) exim4 Mail Server
130 
131 Run the following commands in terminal
132 
133  $ sudo apt-get install exim4
134  $ sudo dpkg-reconfigure exim4-config
135 
136 A Mail Server Configuration window will appear.<br>
137 Follow the following instructions to configure the mail server.
138 
139  1) The first page is just and introduction. Press ENTER
140  2) On the second page choose the second option i.e
141  mail sent by smarthost; received via SMTP or fetchmail and
142  press ENTER.
143  3) Next Keep the system mail name as it is and press ENTER.
144  4) Just Press Enter for the next page.
145  5) The next page asks you to enter IP addresses to listen on
146  incoming SMTP connections. Leave it as it is and Press ENTER
147  6) Even on the next page let the value be the default one and
148  Press ENTER.
149  7) Leave the next page as it is and Press ENTER.
150  8) The next page asks you for IP address or host name of the outgoing
151  smarthost. Enter “smtp.example.com::587″. Where example refers to
152  gmail, yahoo or any other mail service provider and 587 is port number.
153  9) The Next page asks you if you want to hide local mail name in
154  outgoing mail? Choose “No”.
155  10)The Next asks you if you want to keep number of DNS-queries minimal?
156  Choose “No”.
157  11)On the next page choose the delivery method for local mail as
158  mbox format in /var/mail/.
159  12)Next page asks you if you want to split configuration into small
160  files? Choose “No”.
161  13)Next keep root and postmaster mail recipient empty.
162 
163 Now terminal will show that MTA is being restarted.<br>
164 After this is done, run the following command in terminal
165 
166  $ sudo vim /etc/exim4/passwd.client
167 
168 Add following in the file
169 
170  *:USERNAME@example.com:PASSWORD.
171 
172  Where, USERNAME is a valid email address and PASSWORD is password for USERNAME.
173 
174 7) Enable site(bakaplan) using Apache
175 
176 Do as following:
177 
178  1) Open misc/bakaplan.conf file
179  2) Change username(*) with your username and save
180  3) Copy bakaplan.conf to /etc/apache2/sites-available
181  $ sudo cp ~/public_html/cgi-bin/bakaplan/misc/bakpalan.conf
182  /etc/apache2/site-available
183  4) Enable site
184  $ sudo a2ensite bakaplan.conf
185  5) Reload apache
186  $ sudo service apache2 reload
187  6) Now open site on browser
188  localhost/bakaplan
